# Душевный компас

Интерактивный психологический self-check инструмент: два коротких теста — «пульс» (состояние прямо сейчас) и «общая картина» (последнее время) — с визуализацией результата в виде «цветка» и «колеса».

## Особенности

- Работает полностью в браузере, без бэкенда и сервера
- Результат кодируется прямо в URL — можно сохранить или отправить ссылкой
- Итог можно сохранить как картинку (PNG)
- Вопросы построены на принципах валидированных психологических шкал (Bond & Lader, Hardy & Rejeski, Svebak & Murgatroyd, Robins, PERMA)
- Один HTML-файл, без зависимостей и сборки

## Как использовать

Открой [Душевный компас](https://olgaya1982-dev.github.io/dushevny-kompas/dushevny-kompas.html) прямо в браузере — установка не нужна.

Vanilla JS, HTML5 Canvas для визуализации, шрифты Google Fonts (Caveat, Space Mono).
